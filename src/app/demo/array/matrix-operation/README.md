---

## 二、核心算法

### 1. 矩阵置零 O(1) 空间

```typescript
// 用首行首列作为标记位
for (i, j in matrix) {
  if (matrix[i][j] === 0) {
    matrix[i][0] = 0;  // 标记第 i 行
    matrix[0][j] = 0;  // 标记第 j 列
  }
}
```

### 2. 原地旋转 90°

```
转置 + 水平翻转 = 顺时针旋转 90°
```

### 3. 螺旋遍历

```
四指针边界: top, bottom, left, right
每圈遍历后边界向内收缩
```
